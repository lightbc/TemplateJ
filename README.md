# TemplateJ

## 版本支持
【IntelliJ IDEA Ultimate 191+】

## 插件描述
* TemplateJ是一个模板生成解决方案。
* 基于FreeMarker模板引擎，一键解决开发过程中的重复过程。
* 对于MySQL数据库有着较好的兼容性。

## 功能描述
* 提供基于FreeMarker模板引擎的模板生成。
* 提供JAVA WEB接口开发的通用模板。
* 提供模板导入/导出功能。
* 提供重命名模板组/模板文件名称功能。
* 提供自定义模板配置。
* 提供多表多模板同时生成。
* 对mysql数据库有较好的兼容性。
